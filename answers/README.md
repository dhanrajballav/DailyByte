## Naming Convention

- **Folder name**: Use your name in lowercase with hyphens (e.g., `john-doe`, `jane-smith`)
- **File name**: `day<number>-solution.<extension>`
  - Example: `day1-solution.py`, `day2-solution.js`, `day3-solution.cpp`

## Solution Format

Your solution file should include:

1. **Header Comment**: Include your name, date, and problem description
2. **Approach Explanation**: Describe your algorithm/approach
3. **Complexity Analysis**: State time and space complexity
4. **Well-Commented Code**: Explain key logic points
5. **Test Cases**: Include the examples from the problem
